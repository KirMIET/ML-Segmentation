## 🔑 Ключевые особенности

### Kirll-0.91954

- **Ансамбль из двух архитектур** — трансформер (SegFormer) + CNN (U-Net) для разнообразия предсказаний
- **EMA (Exponential Moving Average)** — скользящее среднее весов для лучшей обобщающей способности
- **Смешанная точность (AMP)** — ускорение обучения и экономия памяти
- **Lovász-Softmax Loss** — прямая оптимизация IoU
- **Автоматическая оптимизация весов ансамбля** и поиска порога бинаризации

### Ruslan-0.91746

- **UNet++** — улучшенная U-Net с плотными(skip) связями между уровнями
- **SCSE Attention** — пространственное и канальное внимание в декодере
- **Двухфазное обучение** — разморозка энкодера после прогрева декодера
- **Стратификация по платформам** — равное разбиение по весовыми платформами
- **Адаптивный порог** — подбор порога для каждого фолда на каждой эпохе
